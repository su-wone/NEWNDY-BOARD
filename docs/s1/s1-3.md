## 배경

로그인 이후 팀 단위로 보드를 사용하려면 워크스페이스 단위의 격리가 필요합니다. 팀장이 워크스페이스를 생성하고 팀원에게 초대 링크를 발송하여 합류시키는 플로우를 구축합니다.

---

## 방향성

워크스페이스는 Newndy Board의 최상위 협업 단위입니다. 하나의 워크스페이스 안에 Sprint, Task, 팀원이 모두 귀속됩니다. 초대는 이메일 링크 방식으로 구현하며, 링크 클릭 시 로그인 후 자동 합류됩니다.

---

## 제품 요구사항

### 1. 워크스페이스 생성
- 최초 로그인 시 워크스페이스 생성 온보딩 화면 표시
- 워크스페이스 이름 입력 및 생성 API
- 생성자는 자동으로 Admin 역할 부여

### 2. 이메일 초대
- 팀원 이메일 입력 → 초대 링크 발송 (Nodemailer)
- 초대 링크 유효 기간 7일, 1회 사용 후 만료
- 초대 링크 클릭 → 로그인 → 워크스페이스 자동 합류

### 3. 워크스페이스 합류 처리
- 초대 토큰 검증 및 사용자-워크스페이스 연결 DB 처리
- 합류 후 `/dashboard` 리다이렉트
