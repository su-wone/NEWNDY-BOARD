## 배경

Newndy Board는 팀 워크스페이스 기반 서비스이므로 계정 인증이 모든 기능의 전제 조건입니다. 별도 회원가입 없이 Google 계정으로 즉시 로그인할 수 있도록 하여 온보딩 마찰을 최소화합니다.

---

## 방향성

NestJS의 Passport.js 기반 Google OAuth 2.0 인증을 구현하고, JWT (Access + Refresh Token) 방식으로 세션을 관리합니다. 로그인 여부에 따라 페이지 접근을 제어하는 미들웨어를 함께 구성합니다.

---

## 제품 요구사항

### 1. Google OAuth 2.0 설정
- Google Cloud Console OAuth 앱 등록 (Client ID / Secret 발급)
- NestJS `passport-google-oauth20` + `@nestjs/passport` 설정
- Callback URL 환경별 분리 (local / production)

### 2. JWT 토큰 발급 및 관리
- 로그인 성공 시 Access Token (15분) + Refresh Token (7일) 발급
- Refresh Token DB 저장 및 재발급 로직 구현
- `@nestjs/jwt` 기반 Guard 구현

### 3. 로그인 / 로그아웃 UI
- Next.js 로그인 페이지 (`/login`) — Google 로그인 버튼
- 로그아웃 시 토큰 삭제 및 `/login` 리다이렉트

### 4. 인증 미들웨어
- 비로그인 상태에서 보호된 페이지 접근 시 `/login` 리다이렉트
- 로그인 상태에서 `/login` 접근 시 `/dashboard` 리다이렉트
